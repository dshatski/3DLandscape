3DLandscape

C# XNA 4.0-based free flight viewer with shaders for reflection/refraction

View the Project on GitHub dshatski/3DLandscape

3D terrain viewer based on Riemers XNA tutorials updated to work with XNA 4.0 and newer shaders. Features:

  • First-person mouse view and keyboard movements
  • Multitexturing based on view distance
  • Bump-mapped water with shader-driven refractions and specular reflections
